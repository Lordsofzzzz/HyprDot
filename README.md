# HyprDot

Managed with [GNU Stow](https://www.gnu.org/software/stow/).

## Prerequisites

- [Quickshell](https://quickshell.org/) — Qt-based Wayland shell
- [Hyprland](https://hyprland.org/) — Wayland compositor
- [Alacritty](https://alacritty.org/) — terminal emulator
- [matugen](https://github.com/InioX/matugen) — Material You color generation
- A [Nerd Font](https://www.nerdfonts.com/) (e.g. FiraCode Nerd Font)

## Install

```bash
git clone git@github.com:Lordsofzzzz/HyprDot.git ~/HyprDot
cd ~/HyprDot
stow .
```

This symlinks all configs to their expected locations under `~/.config/`.

## What's included

### `quickshell/`
Custom Quickshell shell with Wayland-native widgets:
- **Bar** — workspaces, clock, system tray, audio, bluetooth, network, battery, CPU, backlight
- **Launcher** — app launcher with keyboard navigation and fuzzy search (toggle via Hyprland global shortcut)

### `hypr/`
Hyprland config in Lua:
- `hyprland.lua` — keybinds, monitors, animations, window rules
- `hyprland-colors.lua` — generated by matugen

### `alacritty/`
Terminal config with Catppuccin-inspired ANSI palette.

### `matugen/`
Material You theming engine config:
- `config.toml` — maps templates to output files
- `templates/` — templates for quickshell, alacritty, and hyprland colors

Regenerate colors after changing wallpaper:
```bash
matugen -j image ~/path/to/wallpaper
```

### `scripts/`
- `wallpaper-picker.sh` — picks a random wallpaper and runs matugen

## Workflow

```bash
# Edit configs as usual (symlinks point to the repo)
vim ~/.config/quickshell/Launcher.qml

# Commit changes
cd ~/HyprDot
git add .
git commit -m "change something"
git push
```
