# HyprDot

Hyprland dotfiles managed with [GNU Stow](https://www.gnu.org/software/stow/).

Each top-level directory is a standalone Stow package that symlinks into `$HOME`.

## Prerequisites

- [Hyprland](https://hyprland.org/) (0.55+ for Lua API) — Wayland compositor
- [Quickshell](https://quickshell.org/) — Qt-based Wayland shell (bar, launcher, system tray)
- [Alacritty](https://alacritty.org/) — terminal emulator
- [matugen](https://github.com/InioX/matugen) — Material You color generation from wallpaper
- [swaybg](https://github.com/swaywm/swaybg) — wallpaper display
- [grim](https://sr.ht/~emersion/grim/) + [slurp](https://github.com/emersion/slurp) — screenshots
- [wpctl](https://github.com/PipeWire/wireplumber) — audio control
- [brightnessctl](https://github.com/Hummer12007/brightnessctl) — backlight control
- [playerctl](https://github.com/altdesktop/playerctl) — MPRIS media control
- A [Nerd Font](https://www.nerdfonts.com/) (e.g. FiraCode Nerd Font)

## Install

```bash
git clone git@github.com:Lordsofzzzz/HyprDot.git ~/HyprDot
cd ~/HyprDot

# Deploy all packages
stow hypr alacritty quickshell matugen scripts

# Or deploy individually
stow hypr        # Hyprland compositor config
stow alacritty   # Terminal config
stow quickshell  # Wayland shell (bar, launcher, wifi panel)
stow matugen     # Material You theming config
stow scripts     # Wallpaper picker → ~/.local/bin/
```

## What's included

### `hypr/`

Hyprland compositor config in Lua (revived API, not the legacy `hyprlang` format):

- `hyprland.lua` — monitors, animations (15 animation leaves), keybinds, window rules, input config
- `hyprland-colors.lua` — dynamic border colors, generated by matugen

### `quickshell/`

Custom Wayland shell with system integration. See [`shell.qml`](quickshell/.config/quickshell/shell.qml):

- **Bar** — workspaces (dynamic visibility), system tray, bluetooth, network, audio (PipeWire), CPU, battery (UPower), backlight, clock
- **Launcher** — desktop entry app launcher with keyboard navigation and fuzzy search. Bound to `SUPER+Space` via Hyprland global shortcut.
- **WiFi** — network manager panel with password entry, bound to `SUPER+W`.

Color values are exposed via a generated singleton `Colors.qml` (regenerated by matugen on wallpaper change).

### `alacritty/`

Terminal config with Catppuccin Mocha ANSI palette and a Material You background/foreground (generated by matugen).

### `matugen/`

Material You theming engine — maps templates to output files:

| Template | Generates |
|---|---|
| `templates/quickshell.qml` | `~/.config/quickshell/Colors.qml` |
| `templates/alacritty.toml` | `~/.config/alacritty/alacritty.toml` |
| `templates/hyprland-colors.lua` | `~/.config/hypr/hyprland-colors.lua` |

Regenerate colors after changing wallpaper:

```bash
matugen image ~/path/to/wallpaper --prefer saturation
```

Then reload Hyprland and restart Quickshell (or use `wallpaper-picker.sh` which does all of this automatically).

### `scripts/`

- `wallpaper-picker.sh` — picks a wallpaper and runs matugen, reloads Hyprland, restarts Quickshell
  - No arguments: opens `rofi` dmenu to browse wallpapers in `~/Downloads`
  - `-r` / `--random`: picks a random wallpaper
  - `-s <path>`: sets a specific image
  - `--restore`: restores the last wallpaper from cache (used on boot autostart)

## Keybinds

| Key | Action |
|---|---|
| `SUPER+Space` | Toggle launcher |
| `SUPER+W` | Toggle WiFi panel |
| `SUPER+Q` | Open Alacritty |
| `SUPER+C` | Close focused window |
| `SUPER+E` | Open Nautilus |
| `SUPER+M` | Logout |
| `SUPER+S` | Toggle scratchpad (special workspace "magic") |
| `SUPER+SHIFT+S` | Move window to scratchpad |
| `SUPER+V` | Toggle float |
| `SUPER+P` | Pseudo-tile |
| `SUPER+J` | Toggle split |
| `SUPER+1..0` | Switch to workspace N |
| `SUPER+SHIFT+1..0` | Move window to workspace N |
| `Print` | Full screenshot |
| `SUPER+Print` | Region screenshot |
| `SHIFT+Print` | Active monitor screenshot |
| `SUPER+arrow` / `hjkl` | Focus direction |
| `XF86Audio*` | Volume, media, backlight control |

```bash
# Edit configs as usual (symlinks point to the repo)
vim ~/.config/quickshell/Launcher.qml

# Commit changes
cd ~/HyprDot
git add .
git commit -m "change something"
git push
```
